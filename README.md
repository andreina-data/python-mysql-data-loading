Este proyecto tiene como objetivo automatizar la creación y carga de una base de datos relacional en MySQL utilizando Python. 
Se desarrolló un proceso para crear las tablas, importar información desde archivos CSV y ejecutar consultas SQL sobre los datos almacenados.

Se utilizó Python 3, MySQL, MySQL Workbench "mysql-connector-python" y Módulo "csv"

Conclusiones: Este proyecto permitió reforzar la integración entre Python y MySQL, aplicando conceptos de modelado de bases de datos, carga automatizada de información y ejecución de consultas SQL. Además, se fortalecieron las habilidades en el manejo de datos y la automatización de procesos mediante Python.
